Data Science Project: California Housing Price Prediction using Multivariate Regression Model

April 13, 2020

The objective of this project is to predict future house price using existing California housing data using multivariate regression model. To achieve this goal, we have collected housing data data src, which contains the block groups (or say districts) in California and some statistics about them from 1990 census data. This data has metrics such as population, households, median income, median housing price, and so on. Distances among the centroids of each block group was computed as measured in latitude and longitude. The data contained 20,640 observations on 10 variables. The dependent or target variable is median house value. Then we cleaned and preprocessed the raw data using pandas library. Then, we have performed supervised learning to develop a powerful ML model. We use Regression model to predict the future house price. As house price depends on many parameters, we build multivariate regression model. We have used Train/Test Split to split the data for training and testing. Then trained our model with train data set and evaluate the model against our test dataset.

The columns are: longitude, latitude, housing_median_age, total_bedrooms, population, households median_income, median_house_value, ocean_proximity.

3. Pre-process the data

In the dataset, the value of median_income is continuous. It needs to be converted into categorical type in order to properly split the entire data into train set and test set. We performed data stratification, i.e., the data set is divided into homogeneous subgroups called strata. In statistics, stratification is the process of dividing the rows of a data frame into subgroups that have common properties. You should not have too many strata, and each stratum should be large enough. We add a new categorical feature income_cat to our housing data.

The following code creates an income category attribute by dividing the median income by 1.5 (to limit the number of income categories or strata, the max median_income is 15 in this case), and rounding up using ceil (to have discrete categories), and then merging all the categories greater than 5 into category 5.

df["income_cat"] = np.ceil(df["median_income"]/1.5)
df["income_cat"].where(df["income_cat"] < 5, 5.0, inplace=True)

4. Creating train and test dataset

We will split entire dataset into train and test dataset based on income categories. There are two ways to split dataset: (1) using train_test_split from sklearn.model_selection, and (2) using StratifiedShuffleSplit from sklearn.model_selection.

Since we derived a new attribute “income_cat” by data stratification, we will use StratifiedShuffleSplit technique for splitting the data.

# 80% for traning, 20% for testing
from sklearn.model_selection import StratifiedShuffleSplit
split_df = StratifiedShuffleSplit(n_splits=1, test_size=.2, random_state=4)

# Calculate train and test index
for train_index, test_index in split_df.split(df, df["income_cat"]):
    train_set = df.loc[train_index]
    test_set = df.loc[test_index]

# Removing income_cat field from training and test data sets to go back to original state.
train_set.drop(["income_cat"], axis=1, inplace=True)
test_set.drop(["income_cat"], axis=1, inplace=True)

9. Creating pipeline

  1. Define the individual components of the pipeline
  2. Combine these individual components using sklearn.pipeline.FeatureUnion()

Create a numerical pipeline with following components:

  • DataFrameSelector - to select the attributes
  • StandardScaler - to bring all attributes to the same level
# Create numerical pipeline
from sklearn.impute import SimpleImputer

num_pipeline = Pipeline([('selector', DataFrameSelector(num_attr)),
                        ('std_scaler', StandardScaler())])

Create a categorical pipeline with following components:

  • DataFrameSelector - to select the attributes
  • CategoricalEncoder - to convert categorical labels to numerical labels
# Create categorical pipeline
cat_pipeline = Pipeline([('selector', DataFrameSelector(cat_attr)),
                        ('cat_encoder', CategoricalEncoder())])

Create full pipeline by concatenating all pipelines using FeatureUnion.

# Create full pipeline by concatenating all pipelines using FeatureUnion
dp_pipeline = FeatureUnion(transformer_list=[
    ('num_pipeline', num_pipeline),
    ('cat_pipeline', cat_pipeline),
])

Prepare entire training data set using pipeline.

x_train_prepared = dp_pipeline.fit_transform(x_train)
